# 屎山指数 / Shit Mountain Index

Shit Mountain Index，简称 `SMI`，是一套确定性、可解释、按语言测绘的代码异味启发式。

核心层负责展品发现、总分封顶、危险等级与稳定排序；Java 和 Python 适配器分别理解自己的语法。系统不调用外部服务，也不使用黑箱 AI。

[返回项目首页](../README.md) · [浏览完整展品目录](EXHIBITS.md)

## Java 适配器

| 信号 | 计分 | 单项上限 |
|---|---:|---:|
| 决策点：`if`、`for`、`while`、`case`、`catch`、逻辑操作符 | 每处 `+2` | `24` |
| 块嵌套深度 | 每层 `+4` | `16` |
| 魔法数字：排除 `-1`、`0`、`1` | 每处 `+2` | `20` |
| 含糊命名：单字母变量、非 PascalCase 公共类 | 每处 `+1` | `10` |
| 超长方法：超过 25 行 | 每 5 行 `+2` | `16` |
| 全局可变字段 | 每处 `+6` | `18` |
| 捕获 `Exception` / `Throwable` 与空 catch | 每处 `+8` / `+6` | `20` |
| 可疑注释 | 每处 `+5` | `15` |
| 重复语句 | 每次重复 `+3` | `15` |

## Python 适配器

Python 使用标准库 `ast` 与 `tokenize`，不复用 Java 的文本规则。

| 信号 | 计分 | 单项上限 |
|---|---:|---:|
| 决策点、布尔分支、推导式与模式分支 | 每处 `+2` | `24` |
| 控制结构嵌套深度 | 每层 `+4` | `16` |
| 魔法数字：排除 `-1`、`0`、`1` | 每处 `+2` | `20` |
| 可变默认参数 | 每处 `+30` | `30` |
| 模块级可变状态 | 每处 `+8` | `24` |
| `except Exception` / `BaseException` 与裸 `except` | 每处 `+10` / `+12` | `24` |
| 超长函数：超过 25 行 | 每 5 行 `+2` | `16` |
| `eval` / `exec` 动态执行 | 每处 `+20` | `40` |
| 可疑注释 | 每处 `+5` | `15` |
| 重复语句 | 每次重复 `+3` | `15` |

各项相加后展示分数封顶为 `100`。不同语言可以拥有不同指标和权重，但规则必须保持稳定、透明并由回归测试锁定。
